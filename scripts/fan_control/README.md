# TrueNAS Fan Speed Control Script

## Overview

This script provides advanced fan speed control for TrueNAS SCALE servers, allowing dynamic and intelligent fan management based on system temperatures and load.

## Features (Planned)

- Temperature-based fan speed control
- Logging of fan speed changes
- Configurable temperature thresholds
- Support for multiple fan zones/groups

## Requirements

- TrueNAS SCALE 25.10.1 or later
- Python 3
- ipmitool (for IPMI-based fan control)

## Usage

```bash
# Planned usage example
./fan_speed_control.py --mode auto
```

## Configuration

Configuration will be managed via a JSON or YAML config file with options like:
- Minimum/maximum fan speeds
- Temperature thresholds
- Logging preferences
